# Быстрая настройка Telegram бота

## Что нужно сделать

### 1. Получить ваш Chat ID

**Самый простой способ:**
1. Откройте Telegram
2. Найдите бота `@userinfobot`
3. Отправьте `/start`
4. Скопируйте ваш Chat ID (число, например: `123456789`)

### 2. Настроить переменную окружения

**Для локальной разработки:**
Создайте файл `.env.local` в папке `viktorija-web`:
```env
VITE_TELEGRAM_ADMIN_CHAT_ID=ваш_chat_id
```

**Для Vercel:**
1. Vercel Dashboard → Settings → Environment Variables
2. Добавьте: `VITE_TELEGRAM_ADMIN_CHAT_ID` = ваш Chat ID
3. Выберите все окружения (Production, Preview, Development)
4. Сохраните и пересоберите проект

### 3. Готово!

Теперь бот будет отправлять уведомления о:
- ✅ Новых регистрациях
- ✅ Записях на уроки
- ✅ Запросах к психологу

## Проверка работы

1. Зарегистрируйтесь через Google
2. Проверьте Telegram - должно прийти уведомление

Если не работает:
- Проверьте консоль браузера (F12)
- Убедитесь, что Chat ID правильный
- Убедитесь, что вы отправили сообщение боту хотя бы раз

Подробная инструкция: `TELEGRAM_BOT_SETUP.md`
