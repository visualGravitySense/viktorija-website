Я помогу вам создать программу для SEO-анализа конкурентов и получения рекомендаций для рекламных кампаний.

Вот комплексное решение на Python с использованием различных инструментов:Я создал для вас комплексную программу для SEO-анализа конкурентов. Вот что она делает:

## Основные возможности:

1. **Сбор SEO данных**: title, meta description, заголовки H1/H2, ключевые слова, количество слов, ссылки, изображения
2. **Анализ нескольких конкурентов** одновременно
3. **AI-рекомендации** через Claude API для настройки рекламных кампаний
4. **Генерация отчетов** в JSON формате

## Установка зависимостей:

```bash
pip install requests beautifulsoup4 anthropic
```

## Использование:

1. Получите API ключ от Anthropic на https://console.anthropic.com/
2. Установите его в коде или через переменную окружения `ANTHROPIC_API_KEY`
3. Укажите URL конкурентов
4. Запустите анализ

## Альтернативы для более глубокого анализа:

- **Scrapy** - для масштабного краулинга
- **SEMrush/Ahrefs API** - для профессиональных SEO метрик
- **Google Search Console API** - для анализа собственного сайта
- **OpenAI GPT-4** вместо Claude (если предпочитаете)
